# Vigilant 🚨  
### AI-Powered Smart Surveillance & Piracy Detection Platform 

Vigilant is an AI-powered surveillance and piracy detection platform that uses intelligent video fingerprinting, cloud storage, and real-time analysis to identify duplicate or suspicious video content.  

The system is designed to help content creators, organizations, and security systems detect unauthorized video usage and monitor threats efficiently using modern AI and cloud technologies.

---

# ✨ Features

- 🎥 Video Upload & Registration  
- 🔍 AI-Based Video Fingerprinting  
- ⚡ Real-Time Video Analysis  
- ☁️ Cloud Storage Integration  
- 📊 Interactive Dashboard  
- 🧠 Gemini AI Integration  
- 📁 Threat Activity Logs  
- 🖼️ Automatic Thumbnail Generation  
- 🔐 Secure Cloud-Based Backend  
- 📱 Responsive Modern UI  
- 🚨 Piracy Detection & Threat Monitoring
